Run this chat application by first typing "python ChatServer.py [port number]" into a terminal 
window to start up the server. Then, in another terminal window, type "python ChatClient.py 
[IP address] [port number] [nickname] [ID number] to start a client. Open as many more terminal 
windows as desired to run more clients. To exit, type "disconnect" into the client terminals
and control-C out of the server terminal.
